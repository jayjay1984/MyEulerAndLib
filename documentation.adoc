1+2+3+4+...+n = \frac{(n*(n+1))}{2} \\
S = 3 * \sum_{i=1}^{333}i + 5*\sum_{k=1}^{200}k = 3 * \frac{333*(333+1)}{2} + 5 * \frac{200*(200+1)}{2} = 266333
